# vimrc
VIM configuration file with python autocomplete


***Dependencies:***
```bash
  pip install jedi
```

*** Use: ***

1. clone repository
2. copy vimrc to ~/.vimrc
3. Launch vim and run :PluginInstall
4. To install from command line: vim +PluginInstall +qall
